## Overview

**Crypto Journal App (FARPS)** combines the power of a real-time **FastAPI backend** and a responsive **React frontend** to give traders a personal system for:
- Recording trades and market observations  
- Reviewing emotional and technical performance  
- Analyzing trends with clean data visualizations  
- Refining strategies through reflection and statistics
